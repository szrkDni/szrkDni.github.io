# Profilkép feltöltése

Kérlek, töltsd fel a profilképedet `profile.png` néven ebbe a mappába.

A kép formátuma lehet: `.jpg`, `.jpeg`, vagy `.png`
A kép neve legyen: `profile.png`

Alternatíva: Ha más néven szeretnéd feltölteni, módosítsd a `components/hero.tsx` fájlban a kép elérési útját:

```tsx
<Image
  src="/your-image-name.png"
  alt="Profile"
  ...
/>
```
